# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-02-02

### Added

- Introduced `ValidationErrorCode` enum to standardize validation errors
- Added error code support to `ValidationResult`

## [1.2.0] - 2026-01-29

### Added

- Validate CPF verification digits using check digit algorithm
- Tests for valid and invalid CPF verification digits

## [1.1.0] - 2026-01-29

### Added

- CPF validation for repeated digits (e.g. 11111111111)

## [1.0.0] - 2026-01-23

### Added

- Initial release
- CPF basic validation
- Name validation
